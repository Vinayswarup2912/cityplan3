Course Overview - Algorithms

1. Problems in Nature

Iteration
  • Repeats steps in a loop until the job is done.
  • Examples: The sun rising and setting, counting tree rings.
Recursion
  • Solves a problem by breaking it into smaller steps.
  • Examples: Tree branches, factorials (5! = 5 × 4 × 3 × 2 × 1).

2. Space and Time Efficiency

Space Efficiency

How much memory an algorithm uses. Fewer resources are better.

Time Efficiency

How fast an algorithm runs. Faster performance leads to better outcomes.

Optimized Performance: Faster processing

Efficient Memory Use: Saves memory

Real-World Impact: Improves system reliability

Resource Constraints: Smooth operation on limited devices
